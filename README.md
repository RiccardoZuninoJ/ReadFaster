### Use the tool
[readfaster.riccardozunino.ninja](https://readfaster.riccardozuninoj.ninja)

#### Installation

To install the project on your local machine:

    npm install

Then, to run it:

    npm run dev

### Dependencies 
@react-pdf/renderer
next/font/google
